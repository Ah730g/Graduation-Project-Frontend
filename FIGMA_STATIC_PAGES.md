## How to Use

1. **Access the pages directly** using the URLs above - no authentication required
2. **All pages contain real mock data** that matches the page content
3. **All images use files from the Naruto folder** in the public directory
4. **Pages are fully functional** for viewing and can be converted to Figma designs

## Notes

- All pages are accessible without login when accessed via `/figma` routes
- Mock data is automatically loaded when accessing these routes
- The pages use the same components as the regular application
- All API calls are intercepted and return mock data automatically
- Images are sourced from `/Naruto/` folder in the public directory
